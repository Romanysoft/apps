Support

If you have any questions or problems concerning Multi Commander look first in the online documentation or at the FAQ.

For any bugs, questions, suggestions and feature requests you can use the online Issues Form for private help.
If you want faster response, visit the online support forum first.
You can also use the "Send Feedback" option from inside Multi Commander. It is found under the 'Help' Menu

If you need to send a private message to the author for any other reason then you should use the contact form.
